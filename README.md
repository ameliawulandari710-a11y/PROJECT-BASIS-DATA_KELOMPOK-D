## Deskripsi
Perencanaan dan Implementasi Basis Data Pada Sistem Reservasi Hotel Menggunakan MySQL merupakan proyek yang berfokus pada perancangan dan pembangunan sistem basis data relasional untuk mengelola proses reservasi hotel secara terintegrasi. Sistem ini mencakup pengelolaan data tamu, kamar, tipe kamar, reservasi, check-in, check-out, fasilitas, serta transaksi pembayaran dengan menerapkan konsep ERD, normalisasi hingga 3NF, dan implementasi objek basis data lanjutan pada MySQL.

## Cara Menjalankan Project

1. Jalankan file `01_ddl.sql` untuk membuat database dan tabel.
2. Jalankan file `02_dml.sql` untuk mengisi data awal.
3. Jalankan file `03_query.sql` untuk menjalankan query yang digunakan dalam sistem.
4. Jalankan file `04_view.sql` untuk membuat view.
5. Jalankan file `05_procedure.sql` untuk membuat stored procedure.
6. Jalankan file `06_trigger.sql` untuk membuat trigger.
7. Jalankan file `07_function.sql` untuk membuat function.
8. Alternatif lain, database dapat langsung digunakan dengan melakukan restore file `database_dump.sql`.
